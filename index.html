Troubleshooting

‚ö†Ô∏è Phone cannot find PC (Stuck on Scanning)

This is usually a Windows Firewall issue.

  1. Press Windows Key and type "Allow an app through Windows Firewall".
  2. Click Change Settings.
  3. Find UseAsServer.exe or python in the list.
  4. Check BOTH boxes: ‚òë Private and ‚òë Public.
  5. Click OK and restart the server.
üé§ Microphone hears my voice but PC doesn't

The app sends audio to your PC's "Cable Input". You need to route it to your chat app.

Fix: In Zoom/Discord settings, set Microphone Input to "CABLE Output (VB-Audio Virtual Cable)".

ü§ñ Voice sounds robotic or high-pitched

This is a sample rate mismatch.

The app is optimized for 16000 Hz (Voice Quality) to ensure low latency. The latest server update handles this automatically. Please ensure you are running the latest version of the server.

üéÆ Gamepad isn't working in games

Ensure the ViGEmBus driver was installed correctly during setup.

If not, download it manually here: ViGEmBus Installer.

Note: Connect the gamepad in the app BEFORE launching your game.
